# Oumi

Oumi is a platform to control the state of your debts.

### Before Dev

```bash
yarn install

yarn build

yarn lint

yarn test
```

### Dev

```bash
yarn pm2 up

yarn pm2 proxy:start

yarn watch:packages # terminal
```
